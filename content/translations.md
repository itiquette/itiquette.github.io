## Translation of Programs and Related Writings

For those new to software translation, it can be challenging to find where to start. There's no single portal where all translations happen - instead, the work occurs across various communities and projects. If you're interested in translating free and open-source software, this page will give you links to a lot of pages from where you can start.

[For Swedes Only - TP-SV](http://www.tp-sv.se/): Bookmark this, A friendly Swedish old school mailing list with a lot of history. While it primarily focuses on projects from the Translation Project, members are happy to provide feedback on nearly any Swedish translation. Perfect for beginners who need guidance or have questions.

*Examples: GNU Coreutils, GNU Grep, GNU Tar, Wget, Nano editor*

## Translation Platforms
---

These platforms host multiple projects and provide tools specifically designed for translation work:

- **[Transifex][1]**: A translation platform hosting thousands of projects from tiny utilities to major applications.  
  *Examples: VLC Media Player, GIMP, OBS Studio, Krita, Nextcloud*

- **[Weblate][2]**: An open-source translation platform with a clean interface. Smaller than Transifex.  
  *Examples: phpMyAdmin, FreeCAD, Godot Engine, Jellyfin, calibre*

- **[Translation Project][3]**: One of the oldest translation coordination efforts, focusing primarily on GNU and other core free software.  
  *Examples: GNU Coreutils, GNU Grep, GNU Tar, Wget, Nano editor*

- **[Crowdin][5]**: Translation platform hosting many open source projects.  
  *Examples: OBS Studio, LineageOS, Pi-hole, Ventoy, Mastodon*

## Operating Systems and Desktop Environments
---

### Linux Distributions


- **[Debian][6]**: Offers detailed statistics and information about translation needs across the Debian ecosystem, which influences many other distributions like Ubuntu and Linux Mint.  
  *Examples: APT package manager, Debian Installer, Debian documentation, package descriptions*

- **[Ubuntu][7]**: This guide explains how to join Ubuntu's translation community. Note that many Ubuntu components are actually translated in upstream projects like GNOME or Debian.  
  *Examples: Ubuntu Software Center, Ubuntu Settings, Ubuntu Documentation, Ubiquity installer*

- **[Fedora Weblate][8]**: Translation platform for Fedora Linux and related projects. Good starting point for Red Hat-related translations.  
  *Examples: Fedora Workstation, Fedora documentation, DNF package manager, Anaconda installer*

- **[openSUSE][9]**: Translation platform for this major Linux distribution, with instructions for getting started.  
  *Examples: YaST installer, openSUSE documentation, distribution-specific tools*

- **[Linux Mint][10]**: For translating Linux Mint-specific components. The site clearly marks which translations should happen here versus upstream projects.  
  *Examples: Mint Menu, Mint Update Manager, Mint Welcome Screen, Cinnamon-specific components*

- **[Elementary OS][11]**: Translation portal for this elegant Linux distribution.  
  *Examples: Pantheon desktop environment, AppCenter, System Settings, elementary apps*

- **[Arch Linux Wiki][12]**: While not a traditional translation platform, the Arch Wiki welcomes translations of its extensive documentation into multiple languages.  
  *Examples: Installation guides, software configuration tutorials, troubleshooting guides*

- **[Tails Documentation][13]**: Help translate documentation for this privacy-focused Linux distribution.  
  *Examples: Installation guides, security documentation, usage tutorials*

- **[LineageOS][14]**: Translation portal for this custom Android distribution.  
  *Examples: System UI, settings menu, built-in apps*

### Desktop Environments


- **[GNOME - Damned Lies][15]**: Translation portal for the popular GNOME desktop environment and its many applications. Contains helpful statistics showing translation progress.  
  *Examples: GNOME Shell, GNOME Files (Nautilus), GNOME Calendar, Evince document viewer, GNOME Software*

- **[KDE][16]**: Where you can help translate the KDE desktop environment and its rich ecosystem of applications. Includes good documentation for newcomers.  
  *Examples: KDE Plasma desktop, Dolphin file manager, Konsole terminal, Okular document viewer, Kdenlive video editor*

- **[XFCE Translation Project][17]**: For translating this lightweight and user-friendly desktop environment and its applications.  
  *Examples: XFCE Panel, Thunar file manager, XFCE Terminal, Mousepad text editor*

- **[LXDE][18]**: Guide for translating this lightweight desktop environment, perfect for those interested in making computing accessible on older hardware.  
  *Examples: PCManFM file manager, LXPanel, LXTerminal, LXSession, LXDE settings*

- **[Enlightenment][19]**: Translation portal for this visually distinctive desktop environment and window manager.  
  *Examples: Enlightenment window manager, EFL libraries, Terminology terminal emulator*

## Documentation and Infrastructure Projects
---

- **[GNU Savannah][55]**: The home for many GNU Project translations. If you care about core free software utilities and applications, start here.  
  *Examples: GNU Emacs, GCC, GNU Privacy Guard (GPG), GNU Cash, GNU Health*

- **[Man-pages-l10n][57]**: Project dedicated to translating Linux manual pages (man pages) into various languages. Essential documentation for command-line users.  
  *Examples: System call documentation, library function references, configuration file formats*

- **[OpenStreetMap Translator][58]**: Contribute translations for this collaborative mapping platform.  
  *Examples: Editor interface, map legends, wiki documentation, tagging guidelines*

- [FSFE Translation:](https://fsfe.org/contribute/translators/translators.en.html) Translations regarding FSFE projects.

## Individual Programs
---

Beyond these platforms, consider exploring your favorite programs to see if they need translation help. Look for "Contribute" or "Translation" links on their websites, or check their GitHub/GitLab repositories for translation files.

A good strategy for beginners is to visit [AlternativeTo.net](https://alternativeto.net/) to find popular open-source programs, then check if they have complete translations in your language. Translation processes vary widely - some use platforms like those listed above, while others might ask you to simply email translated files directly to developers.
